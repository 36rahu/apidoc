      no implementation for this version
      Use the OAuth request to retrieve an access token for use with your payments calls. For authentication and authorization related to Identiy, learn how to obtain a user’s consent
      Request:
      Resource:
      https://api.paypal.com/v1/oauth2/token
      Body:
      {
      "grant_type": listGrantType Token grant type(client_credentials)
      }
      Response:
      Body:
      {
      "access_token": string, The access token issued by PayPal.
      "app_id": string,
      "expires_in": number, The lifetime in seconds of the access token. Assigned in response.
      "scope": string, Scopes expressed in the form of resource URL endpoints
      "token_type": string The type of the token issued as described in OAuth2.0 RFC6749
      }
      Sample:
      Request:
      GET /oauth2/token HTTP/1.1 Content-Type: application/json; charset=utf-8 { "grant_type": "client_credentials" }

  • Identity

    Log In with PayPal (formerly PayPal Access) is a commerce identity solution that enables your customers to sign in to your web site quickly and securely using their PayPal login credentials.
